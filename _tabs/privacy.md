본 개인정보처리방침은 **Time Series Lab**(`https://ts.starall.kr`, 이하 "본 블로그")이 방문자의 정보를 어떻게 수집·이용·보관하는지 설명합니다.

최종 수정일: 2026-07-12

## 1. 수집하는 정보

본 블로그는 별도의 회원가입이나 로그인 절차가 없으며, 운영자가 직접 개인정보를 수집·저장하지 않습니다. 다만 아래의 제3자 서비스를 이용하는 과정에서 일부 정보가 자동으로 수집될 수 있습니다.

- **접속 및 이용 기록**: 방문 페이지, 접속 시각, 브라우저·기기 정보, 대략적인 지역, 유입 경로 등
- **쿠키(Cookie)**: 방문자 식별 및 이용 통계, 광고 제공을 위해 사용되는 작은 텍스트 파일
- **댓글 작성 시**: 댓글 기능(giscus)을 통해 GitHub 계정으로 남긴 댓글 및 공개 프로필 정보

## 2. 제3자 서비스

본 블로그는 다음의 외부 서비스를 이용하며, 각 서비스의 정보 처리에는 해당 서비스의 개인정보처리방침이 적용됩니다.

### Google Analytics
방문 통계 분석을 위해 Google Analytics를 사용합니다. Google Analytics는 쿠키를 이용해 방문자의 이용 행태(방문 페이지, 체류 시간 등)를 익명으로 수집합니다. 수집된 정보는 사이트 개선 목적으로만 활용됩니다.

- Google 개인정보처리방침: <https://policies.google.com/privacy>
- Google Analytics 차단(옵트아웃): <https://tools.google.com/dlpage/gaoptout>

### Google AdSense (광고)
본 블로그는 Google AdSense를 통해 광고를 게재할 수 있습니다. Google을 포함한 제3자 광고 사업자는 쿠키를 사용하여 방문자의 이전 방문 기록 등을 바탕으로 맞춤형 광고를 제공할 수 있습니다.

- Google은 광고 쿠키를 사용하여 사용자가 본 블로그 및 다른 사이트를 방문한 기록을 바탕으로 광고를 게재합니다.
- 방문자는 [Google 광고 설정](https://adssettings.google.com)에서 맞춤 광고를 해제할 수 있습니다.
- 제3자 광고 사업자의 쿠키 사용을 해제하려면 <https://www.aboutads.info> 를 참고하세요.
- Google 광고 관련 정책: <https://policies.google.com/technologies/ads>

### giscus (댓글)
댓글 기능은 GitHub Discussions 기반의 giscus를 사용합니다. 댓글을 작성하려면 GitHub 계정 인증이 필요하며, 작성한 댓글과 공개 프로필 정보는 GitHub에 저장됩니다.

- GitHub 개인정보처리방침: <https://docs.github.com/site-policy/privacy-policies/github-privacy-statement>

### GitHub Pages (호스팅)
본 블로그는 GitHub Pages를 통해 호스팅되며, 접속 시 IP 주소 등이 GitHub의 서버 로그에 기록될 수 있습니다.

## 3. 쿠키 관리

방문자는 웹 브라우저 설정을 통해 쿠키 저장을 거부하거나 삭제할 수 있습니다. 다만 쿠키를 차단할 경우 일부 기능 이용에 제한이 있을 수 있습니다. 브라우저별 쿠키 설정 방법은 각 브라우저의 도움말을 참고하시기 바랍니다.

## 4. 개인정보의 보관 및 이용

운영자는 방문자의 개인정보를 직접 보관하지 않습니다. 위에 명시된 제3자 서비스가 수집한 정보의 보관 기간 및 이용 방식은 각 서비스의 정책을 따릅니다.

## 5. 방침의 변경
